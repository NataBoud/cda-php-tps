# Exercice 02

## Objectif

- Maitriser les routes
- Utiliser les vues blade
- Utilisation de layout
- Utilisation de tailwind
- Maitriser le fonctionnement des controllers

## Sujet

Réaliser une application qui permet de gérer une Todo List !

- L'application aura
  - une page d'accueil avec la liste des Todos
  - Une formulaire de création d'un Todo
  - Le détail d'un todo
  - Supprimer un Todo
  - Editer un todo
